# Dirt to Diamond Datapack

## Features

- Custom crafting recipe to convert dirt blocks into diamonds.
- Adds a cool advancement to go along with it.

## Compatibility

- Only tested on Minecraft Java Edition 1.21.4
- Not tested on Bedrock Edition
